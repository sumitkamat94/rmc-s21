Broad area/overview

In this paper authors propose a novel technique which is called hybrid impedance control (HIC). This technique reduces to Khatih's operational space control or Hogan's Impedance control when it simplifies. HIC is a combination of Hybrid Force-Position Control and Impedance Control into one strategy. The paper is offered a hybrid impedance control method based on concept of inner-outer loop control strategy and the proper impedance of the manipulator is calculated based on the environment impedance in each degree of freedom.

The main framework of the HIC is to separate the space as position and force control subspaces and then it is possible to choose the proper impedance of manipulator based on the environment impedance in each subspace.

Notation

  • \(I\): Identity matrix

  • \(J(q)\): Jacobian matrix

  • \(C(q,\dot q)\): Coriolis term (kinetic energy)

  • \(M(q)\): Inertia matrix

  • \(G(q)\): Gravitational matrix(Potential energy)

Specific Problem

HIC is combining previous approaches and novel methods to get a comprehensive plan. In this way inverse dynamics of the system combines with force control methods, i.e. impedance control and hybrid control. This method requires a careful design of the environment to get the proper control startegy.

Solution Ideas

In this method, imedance and hybrid position/force methods are used for building a unified plan of control. Thus, It is possible to find the mechacical impedance of the end-effector while the desired position/force is being tracked. The hybrid impedance control is a parameter dependent control as it includes inverse dynamics. Here let have an example of a study by Akdogan et al [1] where the following relation is proposed for forced based impedence control:

\[\tau = M(q)J(q)({M_d}^{ - 1}[{F_d} - {F_e} - {B_d}\dot x] - \dot J(q)\dot q) + C(q,\dot q) + G(q) + F(q) - J{(q)^T}{F_e}\]

Also, the following relation is calculated for position based impedance control:

\[\tau = M(q)J(q)({\ddot x_d} + {M_d}^{ - 1}[ - {B_d}(\dot x - {\dot x_d}) - {k_d}(x - {x_d}) - {F_e}] - \dot J(q)\dot q) + C(q,\dot q) + G(q) + F(q) - J{(q)^T}{F_e}\]

The hybrid impedance control includes both force based and position based impedance control with a switching matrix S. This matrix is a diagonal matrix with entries either 1 or 0 and used to determine which degrees of freedom of the end effector is controlled by either position based or force based impedance control, respectively.

\[\tau = M(q)J(q)(S{\ddot x_d} + {M_d}^{ - 1}[(I - S){F_d} - {F_e} - {B_d}(\dot x - S{\dot x_d}) - S{K_d}(x - {x_d})] - \dot J(q)\dot q) + C(q,\dot q) + G(q) + F(q) - J{(q)^T}{F_e}\]

By choosing \(M_{d}\), \(B_{d}\), \(k_{d}\) and \(S\) the desired location of the robotic end-effector will be known.

Comments

  • In the simulations, a constant impedance was

maintained, and only force and position commands were adjusted.

  • Compare to real life for instance the Human impedances change constantly, however, and such an approach might prove advantageous in robotics

  • Numerous simplifications were made in the

model. Gear backlash, flexibility, static friction, torque constraints, and noise all detract from the ideal system.

  • It is assumed that a devoted inverse dynamics controller existed which measured force signals, setting up multiple sensors for measuring force and position has its own challenge.
